## STATE MANAGEMENT FLOW

```
useGpsMockDetection Hook
│
├── State Variables
│   ├── detectionResult: null → {...} (structured data)
│   ├── isChecking: false → true → false (loading state)
│   ├── error: null → error message (if any)
│   └── checkCount (ref): 0 → 1 → 2 → ... (for cycling)
│
├── Functions
│   ├── performDetection()
│   │   ├── Sets isChecking = true
│   │   ├── Delay 1.2 seconds (simulates API)
│   │   ├── Calls generateMockedDetectionResult()
│   │   ├── Sets detectionResult = result
│   │   └── Sets isChecking = false
│   │
│   └── generateMockedDetectionResult()
│       ├── Increments checkCount.current
│       ├── Calculates scenario = checkCount % 4
│       ├── Switches on scenario (1/2/3/4)
│       ├── Returns structured result for that scenario
│       └── Result includes ALL needed data
│
└── Effects
    └── useEffect (on mount)
        └── Calls performDetection() once automatically

## SCENARIO LOOKUP TABLE

```
checkCount  │  Scenario  │  Fraud Score  │  Recommendation  │  Status
─────────────┼────────────┼───────────────┼──────────────────┼─────────
0 % 4 = 0   │ Default/4  │  72%          │  REJECT          │  🔴
1 % 4 = 1   │ 1          │  85%          │  REJECT          │  🔴
2 % 4 = 2   │ 2          │  45%          │  MANUAL_REVIEW    │  🟠
3 % 4 = 3   │ 3          │  12%          │  APPROVE         │  🟢
4 % 4 = 0   │ Default/4  │  72%          │  REJECT          │  🔴
5 % 4 = 1   │ 1          │  85%          │  REJECT          │  🔴
...         │ ...        │  ...          │  ...             │  ...
```

Each tap of "Re-check" increments checkCount and cycles through scenarios!
